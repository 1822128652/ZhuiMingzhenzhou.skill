# Assets 目录

这个目录用于存放真昼女朋友skill的资源文件。

## 可以存放什么？

- 📸 **照片** - 你们"一起"的照片、截图等
- 🎁 **礼物图片** - 想要送给对方的礼物图片
- 💌 **情书/便签** - 生成的甜蜜文字图片
- 🎨 **自定义素材** - 真昼的头像、表情包等
- 📝 **模板文件** - 纪念日卡片模板、生日祝福模板等

## 文件格式建议

- 图片: .jpg, .png, .webp
- 文档: .pdf, .txt, .md
- 其他: 任何你想保存的文件

## 使用方式

这些文件不会被自动加载到上下文，但可以在生成回复时引用或使用。
